editor.find_ellipses(src, ksize=1, otsu=True, sortby='size')[source]

Find the rotated rectangle in which the ellipse is inscribed.

Parameters:

  • ksize – size of blur window If a False value is specified, no blurring will be performed.

  • otsu – Otsu’s method is used for thresholding src image. True (r = 1) specifies to use Otsu’s algorithm. A float number (r < 1) specifies threshold percentile.

  • sortby – how to sort (‘pos’ or ‘size’)

Returns:

(cx,cy), (ra,rb), angle

cx, cy : center of the rectangle [pix] ra, rb : ra:width < rb:height of the rectangle [pix] angle : rotation angle in clockwise (from 00:00 o’clock)

Return type:

RotatedRect

Note

To detect small spots, increase the amount of blur. To filter ellipses with rectangle sizes from rmin to rmax: >>> ls = [(c,r,a) for c,r,a in ellipses if rmin < r[0] and r[1] < rmax]
