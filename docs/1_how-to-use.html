2. User Interface

2.1. Maneuvering

For details, see wxpyNautilus’s documentation.

The user interface of wxpyJemacs is the same as wxpyNautilus since they are built on the same package. The only differences are:

  • It has the interface of TEM (REQUEST / NOTIFY).

  • Support GATAN DM image format.
