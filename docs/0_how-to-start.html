1. Getting started

1.1. Prerequisites and Installing

For details, see wxpj/Getting Started

To install PY310+ packages:

$ pip install -U pywin32 httplib2 mwxlib
$ git clone https://github.com/komoto48g/wxpj.git

1.2. First startup

Here <wxpj> is the directory containing the main program.

Launch the program:

$ python <wxpj>/wxpyJemacs.py

To terminate the program, press [x] Button.

Note

Save the session.

When you close the program, a popup window will appear asking “Do you want to save session before closing program?”.

Click [OK]. Then, the next time you start the program with the session file, you can start it in the same state as when it was last closed.

Note

What’s a session?

Like a project file, it roughly saves plugins, parameters, windows layouts, buffers, etc. You can save the session at the exit or any time from the menu [File/Session/Save session].

When you save the sesison, the setting are saved in the session file. The settings will be restored the next time you start the program with the session specified.

1.3. Startup process

When you run the program, it will start with the following steps:

  1. Load siteinit.

    First, it searches for the file in the current directory. If not found, use the one in the <wxpj> directory.

  2. Load startup plugin

    Edit the initial settings. Then, save the current session.

  3. Load the session.

    To restart the session, launch the program with -s switch. Then, the program will start in the same state as when the session was saved:

    $ python <wxpj>/wxpyJemacs.py -s<session file.jssn>

    The extension should normally be “.jssn”.
