3. Image Analysis

3.1. Pitch

3.1.1. Visualization and Validation

If you input an image into the software (black box), it will output numerical values.

Q:

How can we check its accuracy?

A:

With your eyes ^^.

Image processing should not be treated as a black box. It is important to see the results visually.

The basic procedure for image analysis is as follows:

  1. Image processing.

  2. Extract features from processed images.

  3. Evaluate the image features.

  4. Validate results.

3.2. Roundness of DIFF patterns

  1. Load plugins.pragma_ring. The plugin is displaed in [Pragma Tools/Ring] panel.

    Note

    You can load plugins by DnD. The standard modules are included in <wxpj>/plugiuns/.

  2. Pressing [Setting] popups:

    • plugins.lcrf.Plugin [Basic Tools/lcrf] panel,

    • plugins.ld_ring.Plugin [ld_ring] panel.

  3. Press [Run] button. Then, the following steps are performed:

    1. Search center position and fit the model parameters.

    2. Find peaks in radial distribution and plot some markers on the ring pattern.

    3. Caluculate ellipse parameters for fitting the plotted markers.

    If succeeded, the roundness values will be displayed in [ld_ring] panel.

    • [rmin] [rmax] in [Basic Tools/lcrf] set the radius ratio thresholds that give the peak search range (indicated by the two blue lines).

    • Roundness of an ellipse C is evaluated using two ratios: Y/X and Y2/X2, where C is a representative diffraction ring that has the maximum peak in the radial distribution within the search radii.

      真円度は Y/X, Y2/X2 の二つの比で評価される．ここで， 楕円 C は検索半径内において動径方向分布で最大ピークとなる代表円 (もしくは回折リング)．

Tip

To add a marker, select a pixel and press [c]. To remove a marker, select it and press [delete].

cf. Marker-mode bindings:

[n] move-to-next-marker.
[p] move-to-previoius-marker.
[delete] Remove the selected marker.
[escape] Exit marker-mode.

3.3. Distortion of MAG images

  1. Load plugins.pragma_grid. The plugin is displaed in [Pragma Tools/Grid] panel.

  2. Pressing [Setting] popups:

    • plugins.lccf.Plugin [Basic Tools/lccf] panel,

    • plugins.ld_grid.Plugin [ld_grid] panel.

  3. Press [Run] button. Then, the following steps are performed:

    1. Find circles and plot some markers on the center positions.

    2. Caluculate grid parameters for fitting the plotted markers.

    If succeeded, the roundness values will be displayed in [ld_grid] panel.

3.4. Appendix

Tip

The annotation is displayed in the infobar (bottom side of the image).

You can toggle the display of annotations by pressing [S-a].

Tip

You can also use the bufer list view [C-b] to see the annotations.
