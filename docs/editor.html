editor.find_ellipses(src, frmin=None, frmax=None, ksize=1, sortby='size')[source]

Find the rotated rectangle in which the ellipse is inscribed.

Parameters:

  • frmin – min threshold ratio (to src.shape) of ellipses to find

  • frmax – max threshold

  • ksize – size of blur window

  • sortby – key of sorted list (pos or size:default)

Returns:

(cx,cy), (ra,rb), angle

(cx,cy) : center of the rectangle [pix] (ra,rb) : ra:width < rb:height of the rectangle [pix] angle : rotation angle in clockwise (from 00:00 o’clock)

Return type:

RotatedRect
